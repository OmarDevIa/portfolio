Application web pour une maison d'edition

J’ai réalisé cette application web pour un client travaillant sur la génération automatique de légendes d'image à travers un modèle d'intelligence artificielle LLM.

L’objectif ? Permettre de générer rapidement des légendes d'image précises et pertinentes pour une maison d'édition de journal afin d'accélérer le processus de publication.

Mon client, étant novice dans les nouvelles technologies, a la possibilité d'utiliser cette application et d'en tirer les bénéfices suite au déploiement en interne.

Ce projet simple mais fonctionnel a donné entière satisfaction à mon client.

Technologies utilisées : PYTHON, Gradio,huggingface,IBM Cloud,IBM Watsonx réalisé pour le client
